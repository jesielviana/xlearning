Passo à passo para configurar e usar o Xlearning

Gerenciar módulo administrativo

  1. No 1º acesso, para acessar como usuário Adm insira: matricula "1" senha ""12345"
  2. Inseir parâmetro número 1 para path das logos de cursos, ex.: "C://Documentos/xlearning/curso/"
  3. Inseir parâmetro número 2 para path dos avatares de usuários, ex.: "C://Documentos/xlearning/avatar/"
  4. Inseir parâmetro número 3 para path de conteudo, ex.: "C://Documentos/xlearning/conteudo/"
  5. Cadastrar Coodenadores, professores
  6. Cadastrar cursos
  7. Cadastrar disciplinas para os cursos
  8. Criar turmas para curso

Módulo acadêmico

  1. Acessar como professor, consulte os professores para ver sua matrícula
  2. adicionar conteúdo
  3. Acessar como aluno, nesse caso não tem nada ainda par ao aluno, só a index mesmo
